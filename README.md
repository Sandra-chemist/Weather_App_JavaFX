# Weather_App_JavaFX

Technologies:

- Java 11
- JavaFX 19
- SceneBuilder
- CSS
- OpenWeather API

Functionalities:

- current weather conditions (temperature, description, humidity)
- 4-day weather forecast
- weather forecast for two cities at one time
